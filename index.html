Guía de Ejercicios de Matemáticas

Nota: Esta página solo muestra los problemas para lectura y resolución en el cuaderno.

1. Ecuaciones de primer grado

a) Cinco ecuaciones sin signos de agrupación

  1. 5x − 12 = 3x + 8
  2. 7x + 15 = 3x − 9
  3. 9x − 4 = 5x + 20
  4. 11x + 3 = 7x − 21
  5. 18 − 2x = 6x − 10

b) Cinco ecuaciones con signos de agrupación (paréntesis, corchetes y llaves)

  1. 3(x − 4) = 2(x + 5)
  2. 2[3x − 4] = 5(x − 2)
  3. 4{2x − 3[x − 1]} = 12
  4. 5(x − 2) − 3(2x + 1) = 9
  5. 2{3[x + 1] − 2(x − 3)} = 14

3. Ejercicios de intervalos

Intervalos abiertos

  1. Intervalo (2, 7). Representa el intervalo en la recta numérica y determina si 2 y 7 pertenecen al intervalo.
  2. Intervalo (−5, 3). Representa el intervalo en la recta numérica y escribe dos números que pertenezcan al intervalo.

Intervalos cerrados

  1. Intervalo [0, 10]. Representa el intervalo y escribe tres elementos que pertenezcan al intervalo.
  2. Intervalo [−4, 2]. Representa el intervalo y verifica si −4 pertenece al intervalo.

Intervalos semiabiertos

  1. Intervalo [3, 9). Representa el intervalo en la recta numérica y escribe dos números que pertenezcan al intervalo.
  2. Intervalo (−6, −2]. Representa el intervalo en la recta numérica y escribe dos números que pertenezcan al intervalo.

4. Funciones: dominio, codominio, rango y diagramas sagitales

Definiciones

  1. Define con tus palabras: ¿Qué es el dominio de una función?
  2. Define: ¿Qué es el codominio de una función?
  3. Define: ¿Qué es el rango (o imagen) de una función?
  4. Define: ¿Qué son las parejas ordenadas de una función?

Diagramas sagitales que sí son funciones (haz el dibujo)

  1. A = {1, 2, 3}, B = {2, 4, 6}. Define f(x) = 2x y represéntala con un diagrama sagital.
  2. A = {a, b, c}, B = {1, 2, 3}. Dibuja un diagrama sagital donde cada elemento de A se relacione con un solo elemento de B.
  3. A = {4, 5}, B = {1, 2, 3}. Dibuja un diagrama sagital donde 4 → 1 y 5 → 1.
  4. A = {m, n, p}, B = {4, 6, 8}. Asigna a cada elemento de A exactamente una imagen en B y represéntalo.
  5. A = {7, 8, 9}, B = {0, 1, 2}. Dibuja un diagrama sagital que sea función.

Diagramas sagitales que no son funciones

  1. Dibuja un diagrama sagital donde un elemento de A tenga dos flechas hacia dos elementos diferentes de B.
  2. Dibuja un diagrama sagital donde uno de los elementos de A no tenga imagen en B.
  3. Dibuja un diagrama donde se representen flechas desde B hacia A (relación no presentada como función de A en B).

6. Límites de funciones

6.a) Límites por tabla de valores

  1. Calcula, usando una tabla de valores alrededor de x = 2: limx→2 (3x − 1)
  2. Calcula, usando una tabla alrededor de x = 0: limx→0 (x2 + 4)
  3. Calcula, usando una tabla alrededor de x = 1: limx→1 (2x2 − 3x)

6.b) Límites a partir de la gráfica (descripción)

  1. Supón una función f(x) cuya gráfica se acerca al valor 5 cuando x se aproxima a 3 tanto por la izquierda como por la derecha. Escribe el valor de limx→3 f(x).
  2. Supón una función con un “hueco” en x = 1, pero cuya gráfica se acerca al valor 2 por ambos lados. Escribe el valor de limx→1 f(x).
  3. Supón una función que tiene una asíntota horizontal en y = 3 cuando x tiende a −2. ¿Cuál es el valor de limx→−2 f(x)?

6.c) Límites por sustitución directa

  1. limx→4 (x2 − 16)
  2. limx→5 (2x + 1)
  3. limx→0 (7x3)

6.d) Límites por factorización

  1. limx→3 (x2 − 9) / (x − 3)
  2. limx→−2 (x2 + 3x + 2) / (x + 2)
  3. limx→1 (x2 − 1) / (x − 1)
